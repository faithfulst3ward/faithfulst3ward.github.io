Updating the Firmware on the Raspberry Pi 4

Getting Started

The Raspberry Pi 4 shipped with a firmware version that wasn’t fully optimized, and as a result it was prone to thermal throttling when under heavy load. An updated firmware has since come out that fixes the issue, and it’s an easy upgrade as long as your running Raspberry Pi OS. I don’t run Raspberry Pi OS, so I keep a spare MicroSD card with it installed and occasionally check for updates. It’s as simple as booting up with Raspberry Pi OS and running:

 sudo apt update && sudo apt full-upgrade -y

You can check and see what version you’re on by running:

sudo rpi-eeprom-update
